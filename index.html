Using Semantic Tags

On my page, I try to stay away from generic block tags, such as SPAN and DIV and use semantic tags instead - header, nav, footer, figure Use containers as article, aside, section, main to put like minded code together

HR is a horizontal line with a semantic meaning Address element indicates that this is an address
